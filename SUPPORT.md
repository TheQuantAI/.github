# How to get support

Thanks for using TheQuantAI projects! Here's how to get help:

## 📖 Documentation

- **QuantSDK Docs**: [docs.thequantcloud.com](https://docs.thequantcloud.com)
- **API Reference**: [docs.thequantcloud.com/api](https://docs.thequantcloud.com/api/)

## 🐛 Bug Reports

If you've found a bug, please [open an issue](https://github.com/TheQuantAI/quantsdk/issues/new?template=bug_report.md) in the relevant repository.

## 💡 Feature Requests

Have an idea? [Submit a feature request](https://github.com/TheQuantAI/quantsdk/issues/new?template=feature_request.md).

## 💬 Community

- **Discord**: [discord.gg/thequantai](https://discord.gg/thequantai) — real-time chat
- **GitHub Discussions**: Use the Discussions tab in each repo for Q&A

## 📧 Direct Contact

- **General**: [team@thequantai.in](mailto:team@thequantai.in)
- **Security Issues**: [security@thequantai.in](mailto:security@thequantai.in) (see [SECURITY.md](SECURITY.md))
