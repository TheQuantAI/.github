# Security Policy

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Report them via email to **security@thequantai.in**.

You should receive a response within 48 hours.

For project-specific security policies, see the `SECURITY.md` in each repository.

## Scope

This policy covers all repositories under the [**TheQuantAI**](https://github.com/TheQuantAI) GitHub organization.

## Safe Harbor

We support safe harbor for security researchers who make a good faith effort to follow responsible disclosure practices.
