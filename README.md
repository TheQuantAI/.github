# .github — TheQuantAI Community Health & Org Profile

This repository contains [TheQuantAI](https://thequantai.in)'s **organization profile** and **community health files** that apply across all repos in the org.

## What's Inside

| File / Folder | Purpose |
|---------------|---------|
| `profile/README.md` | Organization profile displayed on [github.com/TheQuantAI](https://github.com/TheQuantAI) |
| `CODE_OF_CONDUCT.md` | Contributor Covenant Code of Conduct |
| `SECURITY.md` | Security vulnerability reporting policy |
| `SUPPORT.md` | Support channels and resources |
| `FUNDING.yml` | GitHub Sponsors configuration |

## How It Works

GitHub automatically uses files in this `.github` repository as defaults for all repositories in the **TheQuantAI** organization:

- **`profile/README.md`** → Rendered on the org's public profile page
- **`CODE_OF_CONDUCT.md`** → Inherited by repos that don't have their own
- **`SECURITY.md`** → Org-wide security policy
- **`SUPPORT.md`** → Org-wide support information
